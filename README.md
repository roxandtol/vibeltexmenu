# Vibeltex-Menu
A vibe-coded multi game loader for 2 screen arcade cabintets with a touch screen at the bottom.
---
Multi game loader that plays audio through an ASIO stream (useful for certain arcade cabinet), and has a carrousel game selector selectable using an xbox 360 controller (or an api that emulates it), and using a touch screen.
